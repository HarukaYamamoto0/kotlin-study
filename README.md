## 🎯 About

This is my personal Kotlin learning journey. While everyone's asking AI to write code, I'm here reading the official Kotlin documentation line by line — because **real knowledge comes from the source**.

## 🚀 Getting Started

### Prerequisites
- JDK 25+
- Gradle (wrapper included)

### Run the code
```bash
./gradlew run
```

### Run tests
```bash
./gradlew test
```

### Build
```bash
./gradlew build
```
